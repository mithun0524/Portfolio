## ğŸ¨ Customization

### Colors & Theme
Edit `tailwind.config.cjs` to customize the color scheme and extend the theme.

### Fonts
Google Fonts (Bebas Neue, Montserrat, Space Mono) are loaded in `src/index.css`. 
Update font families in the Tailwind config.

### Content
- **Projects:** Update `src/pages/Projects.tsx` with your project data
- **Skills:** Modify skill categories in `src/pages/Skills.tsx`
- **About:** Edit bio and education in `src/pages/About.tsx`
- **Contact:** Update contact information in `src/pages/Contact.tsx`
